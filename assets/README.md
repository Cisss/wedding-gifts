## Image Tips:

1. **Optimize your images** before uploading:
   - Use https://tinypng.com/ or similar tools
   - Keep file sizes under 500KB for fast loading
   
2. **Recommended formats:**
   - JPG for photos
   - PNG for graphics/QR codes
   
3. **Dimensions:**
   - Card images: 600x400px (3:2 ratio)
   - Hero images: 1920x800px
   - Square images: 800x800px

## File Naming:

Use lowercase, no spaces, use hyphens:
- ✅ `our-house.jpg`
- ✅ `honeymoon-beach.jpg`
- ❌ `Our House.JPG`
- ❌ `honeymoon_beach.jpeg`
